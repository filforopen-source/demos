# Application Library Root

**Purpose:**
The primary source code root for the Flutter application.

**Detailed File Overviews:**
- `firebase_options.dart`: Autogenerated Firebase configuration for all supported platforms (Web, iOS, etc.).
- `main.dart`: The main entry point that executes the `bootstrap` process.

**Subdirectories:**
- `src/`: Encompasses the core logic, UI screens, and features of the app.
- `hive/`: Configuration for local data persistence.

**Dependencies/Relationships:**
- Entry point for the entire presentation layer.
