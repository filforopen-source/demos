# Kiosk Home Tests

**Purpose:**
Comprehensive verification of the Kiosk customer journey.

**Detailed File Overviews:**
- `kiosk_home_bloc_test.dart`: Unit tests for the state machine managing the ordering process (e.g., preference selection, art generation).
- `kiosk_home_view_test.dart`: Widget tests that verify the UI transitions and user interaction flows within the Kiosk's step-based home view.

**Dependencies/Relationships:**
- Validates the logic and UI in `genlatte-ui/lib/src/screens/kiosk/home/`.
