# Routing Tests

**Purpose:**
Verifies the application's complex role-based routing and redirection logic.

**Detailed File Overviews:**
- `redirection_test.dart`: A critical suite of tests that simulates different user authentication states and roles (Barista, Kiosk, etc.) to ensure that the `GoRouterRedirector` correctly enforces path boundaries.

**Dependencies/Relationships:**
- Validates logic in `genlatte-ui/lib/src/core/routing/redirection.dart`.
