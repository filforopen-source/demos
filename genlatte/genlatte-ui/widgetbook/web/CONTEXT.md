# Widgetbook Web Entry

**Purpose:**
Coordinates the web-based delivery of the Widgetbook component library.

**Detailed File Overviews:**
- `index.html`: Web entry point.
- `manifest.json`: Web manifest for PWA support.

**Subdirectories:**
- `icons/`: PWA icon assets.

**Dependencies/Relationships:**
- Serves the `genlatte-ui` component previews to a browser.
