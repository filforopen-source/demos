# Widgetbook Component Previews

**Purpose:**
Defines individual stories and previews for UI components using the Widgetbook framework.

**Detailed File Overviews:**
- `main.dart`: Entry point for the Widgetbook application.
- `*_preview.dart`: Individual preview files for various shared widgets (buttons, cards, sliders, etc.).
- `main.directories.g.dart`: Autogenerated directory mapping for Widgetbook's navigation sidebar.

**Dependencies/Relationships:**
- Previews widgets imported from `genlatte-ui/lib/src/widgets/`.
