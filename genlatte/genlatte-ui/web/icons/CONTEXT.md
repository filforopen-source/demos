# Progressive Web App Icons

**Purpose:**
Standardized icon assets required for the Web build, particularly for PWA manifest compliance and browser tab identification.

**Detailed File Overviews:**
- `Icon-192.png` / `Icon-512.png`: Standard square icons for different resolution requirements.
- `Icon-maskable-192.png` / `Icon-maskable-512.png`: Specific variants for platforms that require maskable icons (e.g., Android).

**Dependencies/Relationships:**
- Referenced by `genlatte-ui/web/manifest.json`.
