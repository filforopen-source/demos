# UI Video Assets

**Purpose:**
Stores high-resolution video assets used for loading screens and background animations in the Flutter application.

**Detailed File Overviews:**
- `Dash_Loading_1080px.webm`: An optimized video file featuring the Dash mascot, used during long-running async operations or initial application load.

**Dependencies/Relationships:**
- Loaded by video player widgets in `genlatte-ui/lib/src/widgets/`.
